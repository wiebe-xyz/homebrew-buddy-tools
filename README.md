# Homebrew Buddy Tools Tap

This is the official Homebrew tap for Buddy Tools - a suite of CLI tools to supercharge your Git and GitHub workflows.

## Installation

```bash
brew tap wiebe-xyz/buddy-tools
brew install prbuddy wtbuddy issuebuddy
```

Or install individually:

```bash
brew install wiebe-xyz/buddy-tools/prbuddy
brew install wiebe-xyz/buddy-tools/wtbuddy
brew install wiebe-xyz/buddy-tools/issuebuddy
```

## Available Formulas

| Formula | Description |
|---------|-------------|
| `prbuddy` | CLI tool for managing GitHub Pull Requests with an interactive terminal UI |
| `wtbuddy` | CLI tool for managing Git worktrees for parallel development |
| `issuebuddy` | CLI tool for managing GitHub Issues with an interactive terminal UI |

## Requirements

- macOS (Intel or Apple Silicon)
- GitHub CLI (`gh`) installed and authenticated
